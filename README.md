# alisa_chess
[draft]

Play chess with Yandex.Alisa by voice

```
Алиса:
- Давайте сыграем в шахматы
Юзер:
- Да
Алиса:
- Е4. Ваш ход
Юзер:
- Е5
Алиса:
- Конь f3. Ваш ход
....
Алиса:
- Мат.
```


Chess engine: stockfish 10, level 1
Uses: https://github.com/niklasf/python-chess
